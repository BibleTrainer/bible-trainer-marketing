# BibleTrainer

Marketing site for [BibleTrainer](https://www.bibletrainer.co.uk).

## Dev

- `npm install`
- `npm run dev`
- Add new features in `src/_data/features.json`

Built on top of [11ty](https://11ty.io).

## Build

- `npm install`
- `npm run build`
- Serve `dist`
